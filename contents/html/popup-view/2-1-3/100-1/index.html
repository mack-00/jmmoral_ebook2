중단원 마무리 정리하기

  • 사람은 왜
    관계적 존재
    일까

    • 다양한 관계
      속에서의 나

      ·기존에 맺고 있던 관계가 멀어지기도 하고, 또 새로운 관계를 맺기도 함

      ·다양한 사람과 폭넓은 관계를 맺으며 살아가고 있음

    • 사람이 관계적
      존재인 까닭

      · 다른 사람에게서 필요한 것을 얻기도 하고 ( 도움 정답 )을 받기도 하기 때문

      · 사람들과의 관계 속에서 행복을 느낄 수 있기 때문

      · 사람들과의 만남 속에서 배우고 ( 성장 정답 )할 수 있기 때문 

      · 사람들과 관계를 맺으며 정체성을 지니기 때문

  • 타인과의
    관계에 필요한
    가치와 덕목에
    는 무엇이
    있을까

    • 타인과의 관계에
      필요한 가치와 덕목

      · ( 존중 정답 ): 상대방을 나와 동등한 존재로 대우하는 것

      · ( 배려 정답 ): 자신의 입장만을 내세우지 않고 상대방의 입장과 마음을 헤아리는 것

      · ( 책임 정답 ): 사람들과의 관계를 유지하고 개선하는 데 필요한 역할을 충실하게 수행 하는 것

    • 타인과 바람직한
      관계를 맺는 방법

      · 상대방의 말을 경청하고, 자신의 생각과 감정을 적절하게 표현하며 ( 소통 정답 )하는 태도

      ·나를 도와주는 많은 사람에게 ( 감사 정답 )하는 태도

      ·상대방의 관점에서 헤아리고 ( 공감 정답 )하는 태도

      ·상대방과 내가 다르다는 점을 이해하고 ( 차이 정답 )를 인정하는 태도
