    • 성의 의미

      ·생물학적 의미: 남녀의 ( 성별 정답 )을 구별하는 신체적·생리적 특징

      ·사회·문화적 의미: 사회적으로 형성되고 ( 학습 정답 )된 성의 특징 

      ·넓은 의미: 생물학적인 성과 사회·문화적 성을 모두 포함하는 성적인 것 전체

    • 성의 가치

      ·생명이 탄생하고 다음 세대를 이어 가게 하는 ( 생산적 정답 ) 가치

      ·성적 활동을 통해 감각적 즐거움을 느끼는 ( 쾌락적 정답 ) 가치

      ·서로에 대한 사랑을 표현하고 책임을 확인함으로써 삶을 공유하는 ( 인격적 정답 ) 가치

      ·성차별적 표현: 성별에 따라 부당하게 차별하는 표현 

    • 성에 대한 편견을 
      극복하기 위한 방법 

      ·성별의 차이를 이해하고 있는 그대로의 모습을 ( 존중 정답 ) 하기

      ·일상생활 속에서 ( 성차별적 정답 ) 요소를 감지하는 능력 기르기

      ·성에 대한 편견을 극복할 수 있도록 돕는 법 제정 등 ( 사회적 정답 ) 차원의 노력
