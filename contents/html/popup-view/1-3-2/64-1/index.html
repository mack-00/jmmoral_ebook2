중단원 마무리 정리하기

  • 고통을 어떻게
    대해야 할까

    • 고통의 의미와 종류

      ·고통: 몸이나 마음의 괴로움과 아픔

      ·고통의 종류: 외부에서 신체에 가하는 물리적인 충격이나 건강 문제 등으로 발생하는 ( 신체적 고통 정답 )과 다른 사람과의 갈등, 자신이 하고자 했던 일을 실패했을 때 느끼는 좌 절 등으로 발생하는 심리적 고통이 있음

    • 고통의 원인

      · ( 내적 원인 정답 ):자신의 선택이나 의지 등과 같은 원인 

      · ( 외적 원인 정답 ):생로병사와 같이 내가 통제할 수 없는 원인

      · ( 개인적 원인 정답 ):감정, 욕구 등과 같은 원인 

      · ( 사회적 원인 정답 ): 다른 사람에게 사랑이나 인정을 받고 싶은 욕구, 사회 구조가 지닌 부조리와 불평등 등과 같은 원인 

    • 고통이 우리 삶에서 
      가지는 의미

      ·자신의 삶을 ( 반성 정답 ):하여 한층 더 발전할 수 있음

      ·자신의 삶을 ( 성찰 정답 ):하고 진정한 가치를 발견할 수 있음

      ·인격적으로 성숙할 수 있음

    • 고통을 대하는
      바람직한 태도

      · ( 회복 탄력성 정답 ):통을 주는 역경과 시련을 피하지 않고 용기 있게 마주하며, 그것을 극 복할 수 있도록 도와주는 내면의 힘으로, 시련이나 고난을 이겨 내고 제자리로 돌아오는 긍정적인 힘을 의미함

  • 마음의 평화를
    위해 무엇을
    해야 할까

    • 마음의 평화가
      가지는 중요성

      ·평화로운 마음 상태를 유지할 때 원만하고 평온한 삶을 살아갈 수 있음

    • 마음의 평화를 
      위한 방법

      · ( 마음 챙김 정답 )판단과 편견을 개입하지 않고 ‘지금, 이 순간’에 주의를 집중하는 것

      · 이미 일어난 일에 대한 생각을 바꾸는 연습을 하는 것
